# A8.netサポートへの問い合わせ（下書き）

## 📞 問い合わせ目的

Parameter Tracking機能がMedia Member契約で利用可能か、公式見解を取得する。

---

## 📝 問い合わせ内容（日本語版）

```
件名: パラメータ計測レポートの利用について

お世話になっております。メディア会員の[氏名を記入]です。

パラメータ計測機能について、以下の状況でご相談させてください。

【実施内容】
- 2025年10月13日より、id1パラメータ付きのアフィリエイトリンクを使用
- 過去30日間で合計9回のクリックを実施
- 日別レポート（新レポートB版）では正常にクリック数が記録されている

【問題】
- パラメータ計測レポート画面（https://media-console.a8.net/report/parameter#）で
  「指定された条件で表示できるレポートデータがありません」と表示される
- 期間を広げたり（過去1ヶ月、過去3ヶ月）、パラメータを指定したりしても表示されない
- 3週間以上経過してもデータが表示されない

【質問】
1. メディア会員契約でパラメータ計測レポート機能は利用可能でしょうか？
2. 利用可能な場合、レポートにデータを表示させるための正しい手順を教えてください
3. パラメータ計測レポートのデータ反映には何時間程度かかりますか？
4. 利用不可の場合、会員別の成果追跡を実現する代替方法はありますか？

お手数をおかけいたしますが、ご教示いただけますと幸いです。
よろしくお願いいたします。
```

## 📎 問い合わせ先

### オプション1: Webフォーム（推奨）

**URL**: https://support.a8.net/a8/as/contact/

**手順**:
1. A8.net管理画面にログイン
2. 上記URLにアクセス
3. お問い合わせフォームに以下を入力:
   - 件名: 「パラメータ計測レポートの利用について」
   - 内容: 上記の問い合わせ内容をコピー&ペースト
   - 氏名、メールアドレスを記入
4. 送信

---

4. 送信
